## **Title:**

**Wanderbnb – Digital Property Listing & Booking Platform**

---

## **1. Problem Statement**

In today’s world, travelers face difficulty in finding reliable and affordable accommodations. Traditional booking systems lack personalization, role-based access (host/user), and real-time interaction.

---

## **2. Proposed Solution**

Wanderbnb is a web-based platform that allows:

* Hosts to create and manage property listings
* Users to explore, view, and book listings
* Role-based dashboards for better experience
* Review system for feedback

---

## **3. Features**

* 🔐 User Authentication (Login/Signup)
* 🏠 Host Dashboard (Create/Edit/Delete Listings)
* 🧳 User Dashboard (View Bookings)
* ⭐ Review System
* 📅 Booking & Cancel Booking Feature
* 🎨 Responsive UI using Bootstrap

---

## **4. Technology Stack**

| Layer          | Technology Used           |
| -------------- | ------------------------- |
| Frontend       | HTML, CSS, Bootstrap, EJS |
| Backend        | Node.js, Express.js       |
| Database       | MongoDB                   |
| Authentication | Passport.js               |
| Others         | Mongoose, Method-Override |
